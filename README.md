## Overview

**DigiTools** is a modern responsive digital product platform built as a **Junior Web Development Project** using **React, Vite, Tailwind CSS, and DaisyUI**. The project focuses on delivering a **user-friendly UI**, reusable component structure, and a smooth browsing experience for premium digital tools.

This project reflects my effort to write **clean code**, organize the UI into reusable sections, and build a visually polished frontend that feels modern, practical, and easy to use.

## ✨ Key Features

- Responsive landing page with a clean and modern layout
- Reusable React component architecture for maintainable frontend development
- Product showcase section powered by local JSON data
- Add-to-cart and remove-from-cart interaction
- Dynamic cart count in the navbar for better user feedback
- Smooth scrolling navigation between major sections
- Pricing cards with clear plan comparison
- Beginner-friendly onboarding section with a 3-step workflow
- Toast notifications for cart actions
- User-focused interface with strong visual hierarchy and responsive design

## 🔧 Installation & Setup

Follow these steps to run the project locally on your machine:

1. Clone the repository

```bash
git clone https://github.com/KhalidSifullahSIam2004/DigiTools.git
```

2. Move into the project folder

```bash
cd DigiTools
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open the local server URL shown in the terminal, usually:

```bash
http://localhost:5173
```

## 📖 How to Use

1. Open the homepage and explore the hero section.
2. Scroll to the product section to browse available digital tools.
3. Add products to the cart from the product cards.
4. Open the cart from the navbar icon to review selected items.
5. Remove items if needed or proceed to checkout.
6. Explore the pricing section and onboarding flow for the full user journey.
